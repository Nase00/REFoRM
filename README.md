REFoRM stack

Very early work-in-progress to build an app foundation using *R*edux *E*xpress *F*alc*o*r *R*eact *M*ongo

## Development
```bash
npm install
npm run dev
```
Or, develop with [hot-reloading](http://webpack.github.io/docs/hot-module-replacement-with-webpack.html)
```bash
npm run hot
```

## Production
```bash
npm run prod
```
